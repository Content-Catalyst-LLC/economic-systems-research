# Data Dictionary

| Variable | Description |
|---|---|
| `sector` | Sector or type of work |
| `output` | Output or value added |
| `hours_worked` | Labor input measured in hours |
| `total_wages` | Total labor compensation |
| `labor_productivity` | Output divided by labor input |
| `wage_share` | Labor compensation divided by output |
| `average_wage` | Total wages divided by labor hours |
| `unit_labor_cost` | Average wage divided by labor productivity |
| `year` | Time period |
| `productivity_index` | Indexed productivity level |
| `wage_index` | Indexed wage level |
| `divergence_index` | Productivity index minus wage index |
| `wage_income` | Household wage income |
| `social_support` | Transfers, public benefits, or social wage support |
| `household_cost` | Cost of housing, food, utilities, transport, and essentials |
| `care_reproduction_cost` | Cost of caregiving and household/social reproduction |
| `adequacy_gap` | Wage income plus support minus household and reproduction costs |
| `paid_work_time` | Paid work hours per day |
| `care_time` | Caregiving hours per day |
| `household_time` | Household maintenance and commuting hours per day |
| `rest_recovery_time` | Rest, recovery, and discretionary time |
| `time_poverty_flag` | Indicator for low rest/recovery time |
| `bargaining_power` | Worker bargaining power index |
| `institutional_support` | Labor-law, union, social-insurance, and public-capacity support index |
| `automation_intensity` | Automation exposure or adoption intensity |
| `employment_effect` | Modeled change in labor demand |
| `wage_share_effect` | Modeled change in wage share |
| `quality_effect` | Modeled change in employment quality |
