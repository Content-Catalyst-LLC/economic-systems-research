# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `period` | Time period |
| `price_level` | General price index |
| `inflation_rate` | Percentage change in price level |
| `sector` | Sector affected by energy or supply shock |
| `energy_cost_change` | Change in energy input cost |
| `wage_cost_change` | Change in wage cost |
| `materials_cost_change` | Change in materials or intermediate-input cost |
| `alpha_energy` | Energy-cost pass-through coefficient |
| `beta_wage` | Wage-cost pass-through coefficient |
| `gamma_materials` | Materials-cost pass-through coefficient |
| `estimated_price_change` | Estimated sector price change |
| `household_group` | Household income or vulnerability segment |
| `income` | Household income |
| `energy_spending` | Household energy spending |
| `household_energy_burden` | Energy spending divided by income |
| `nominal_wage` | Wage in nominal terms |
| `price_level_relative` | Relative price level used for real wage calculation |
| `real_wage` | Nominal wage divided by price level |
| `world_price_change` | Change in world import or commodity price |
| `exchange_rate_effect` | Domestic-currency effect of exchange-rate movement |
| `import_price_change` | World price change plus exchange-rate effect |
| `capital_capacity` | Capital availability index |
| `labor_capacity` | Labor availability index |
| `energy_capacity` | Energy availability index |
| `logistics_capacity` | Logistics availability index |
| `supply_availability` | Critical-input availability index |
| `effective_output_capacity` | Binding minimum of key capacity inputs |
| `market_concentration_index` | Stylized concentration index |
| `margin_expansion` | Margin increase during shock |
| `price_amplification_score` | Market-power amplification score |
| `resilience_score` | Composite resilience score |
