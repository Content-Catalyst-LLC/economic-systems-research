# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `exports` | Total exports |
| `imports` | Total imports |
| `output` | Total output or income |
| `trade_openness` | Exports plus imports divided by output |
| `trade_balance` | Exports minus imports |
| `strategic_import_dependence` | Dependence on imported strategic inputs |
| `export_category` | Export group or product category |
| `export_value` | Export value by category |
| `export_share` | Export category share |
| `export_concentration_index` | Sum of squared export shares |
| `gross_exports` | Gross export value |
| `domestic_value_added` | Domestic value added embodied in exports |
| `domestic_value_capture` | Domestic value added divided by gross exports |
| `lead_firm_control` | Degree of external lead-firm control |
| `technology_control` | Degree of domestic control over technology |
| `commodity_share` | Share of exports from commodities |
| `terms_of_trade_shock` | Terms-of-trade change or pressure |
| `foreign_currency_debt` | Foreign-currency debt exposure |
| `capital_flow_volatility` | Capital-flow volatility indicator |
| `reserve_buffer` | Reserve adequacy or buffer indicator |
| `regional_productivity` | Regional productivity score |
| `infrastructure_depth` | Regional infrastructure score |
| `market_access` | Regional market access score |
| `labor_exposure` | Labor exposure to import competition or offshoring |
| `ecological_burden_score` | Embodied emissions, extraction, waste, or land/water burden score |
| `sustainable_trade_resilience_score` | Composite score for resilient, capability-building trade integration |
