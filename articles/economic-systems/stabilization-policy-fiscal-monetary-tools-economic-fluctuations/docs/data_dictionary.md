# Data Dictionary

| Variable | Description |
|---|---|
| `date` | Observation date |
| `recession_indicator` | NBER-based recession indicator from FRED |
| `policy_phase` | Expansion or recession phase label |
| `unemployment_rate` | Civilian unemployment rate |
| `payroll_employment` | Total nonfarm payroll employment |
| `federal_funds_rate` | Effective federal funds rate |
| `treasury_bill_3m` | 3-month Treasury bill rate |
| `cpi` | Consumer price index |
| `real_gdp` | Real gross domestic product |
| `potential_gdp` | Real potential GDP |
| `real_government_consumption_investment` | Real government consumption expenditures and gross investment |
| `real_gdp_growth_annualized` | Annualized quarterly real GDP growth |
| `output_gap_pct` | Percent gap between real GDP and potential GDP |
| `cpi_inflation_yoy_pct` | Year-over-year CPI inflation |
| `delta_unemployment_rate` | Quarterly change in unemployment rate |
| `delta_federal_funds_rate` | Quarterly change in federal funds rate |
| `gov_spending_growth_yoy_pct` | Year-over-year real government consumption and investment growth |
| `taylor_rule_rate` | Illustrative Taylor-rule-style benchmark rate |
| `policy_rate_gap` | Federal funds rate minus illustrative Taylor-rule benchmark |
