# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Circular or regenerative production scenario |
| `recovered_material` | Material input from recovered, reused, recycled, or remanufactured sources |
| `total_material_input` | Total material input used in production |
| `circularity_ratio` | Recovered material divided by total material input |
| `virgin_material_input` | Total material input minus recovered material |
| `residual_waste` | Material remaining for landfill, incineration, leakage, or disposal |
| `total_throughput` | Total material throughput |
| `waste_reduction_ratio` | One minus residual waste divided by total throughput |
| `actual_product_life` | Actual useful product life |
| `baseline_product_life` | Baseline product life under conventional design |
| `product_life_extension` | Actual product life divided by baseline product life |
| `repairability` | Ease of repair |
| `modularity` | Degree to which parts can be replaced or upgraded |
| `disassembly_score` | Ease of separation for repair, remanufacturing, or recycling |
| `value_retention_score` | Composite score for preserving embodied material, energy, labor, and function |
| `ecological_restoration` | Restoration or regenerative benefit |
| `ecological_degradation` | Degradation caused by production |
| `regenerative_balance` | Restoration minus degradation |
| `rebound_effect` | Additional induced use caused by efficiency, convenience, or lower cost |
| `scale_discipline` | Capacity to reduce absolute throughput rather than only relative intensity |
| `circular_justice_score` | Composite score for fair distribution of circular benefits and burdens |
