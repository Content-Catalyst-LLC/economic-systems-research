# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `consumption` | Household consumption component of aggregate demand |
| `investment` | Business or gross investment component of aggregate demand |
| `government_spending` | Government spending component of aggregate demand |
| `net_exports` | Exports minus imports |
| `output` | Aggregate output or income |
| `potential_output` | Estimated potential output |
| `output_gap` | Output minus potential output, divided by potential output |
| `cycle_phase` | Expansion, slowdown, recession, crisis, recovery, or stagnation |
| `expectations_index` | Forward-looking confidence or expectations index |
| `credit_growth` | Change in credit availability |
| `inventory_pressure` | Inventory accumulation or depletion pressure |
| `income` | Household or sectoral income |
| `debt_service` | Required debt payment burden |
| `debt_burden_ratio` | Debt service divided by income |
| `nominal_debt` | Nominal debt stock |
| `price_level` | Price index used for real debt burden |
| `real_debt_burden` | Nominal debt divided by price level |
| `fiscal_multiplier` | Output response to public spending |
| `automatic_stabilizer_strength` | Strength of built-in income support and tax cushioning |
| `delta_g` | Change in government spending |
| `delta_y` | Modeled change in output |
| `growth_rate` | Output growth rate |
| `benchmark_growth` | Growth rate consistent with stable unemployment in the stylized model |
| `beta_okun` | Okun-style unemployment responsiveness parameter |
| `delta_unemployment` | Modeled change in unemployment |
| `external_shock` | Open-economy shock type |
| `shock_intensity` | Shock magnitude |
| `transmission_score` | Composite macroeconomic transmission score |
| `resilience_score` | Composite score for macroeconomic resilience |
