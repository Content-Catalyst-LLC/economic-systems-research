# Data Dictionary

| Variable | Description |
|---|---|
| `household_group` | Stylized household type |
| `income` | Monthly earned income |
| `transfers` | Public or family transfers |
| `liquid_assets` | Liquid asset buffer available for use |
| `debt_service` | Monthly debt-service obligation |
| `other_fixed_burdens` | Fixed non-discretionary obligations besides debt |
| `rent` | Monthly housing cost |
| `food` | Monthly food cost |
| `transport` | Monthly transport cost |
| `utilities` | Monthly utilities cost |
| `health` | Monthly health cost |
| `other` | Other discretionary or semi-discretionary spending |
| `paid_labor_hours` | Daily paid labor time |
| `care_hours` | Daily care and reproductive labor time |
| `commute_hours` | Daily commute time |
| `household_admin_hours` | Daily household logistics, maintenance, and administrative time |
| `rest_discretionary_hours` | Remaining time after work, care, commute, and household logistics |
| `essentials_spending` | Rent + food + transport + utilities + health |
| `essentials_ratio` | Essentials spending divided by income plus transfers |
| `saving` | Income, transfers, and liquid assets less spending, debt service, and fixed burdens |
| `fragility_ratio` | Essentials, debt, and fixed burdens divided by total command over resources |
| `effective_access` | Effective command over essentials after income, transfers, assets, and debt |
| `time_poverty_flag` | Indicator for low rest/discretionary time |
| `public_goods_support` | Scenario value for public provision reducing private burden |
| `asset_depletion_month` | Month when liquid buffers reach zero under stress |
