# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `nominal_rate` | Nominal interest rate |
| `expected_inflation` | Expected inflation rate |
| `real_rate` | Nominal rate minus expected inflation |
| `inflation_gap` | Inflation relative to target |
| `output_gap` | Output relative to potential |
| `neutral_rate` | Reference neutral rate |
| `policy_rate_rule` | Implied policy rate from reaction function |
| `borrower_group` | Household, firm, or public borrower segment |
| `income` | Income, revenue, or fiscal base |
| `debt_service` | Debt service payment |
| `debt_service_ratio` | Debt service divided by income |
| `repricing_share` | Share of debt exposed to near-term rate changes |
| `credit_spread` | Spread over risk-free or policy benchmark |
| `equity_price_change` | Equity-market movement |
| `exchange_rate_pressure` | Currency pressure variable |
| `lending_tightness` | Lending standards or underwriting tightness |
| `financial_conditions_index` | Composite restraint score |
| `cash_flow` | Future cash flow in valuation scenario |
| `discount_rate` | Rate used for present-value calculation |
| `duration_years` | Maturity or valuation horizon |
| `liquid_assets` | Bank liquid asset stock |
| `short_term_outflows` | Potential short-term liquidity outflows |
| `liquidity_coverage_ratio` | Liquid assets divided by outflows |
| `public_debt` | Public debt stock |
| `average_interest_rate` | Effective average public borrowing cost |
| `public_debt_service` | Interest cost on public debt |
| `investment_area` | Long-horizon public or productive investment category |
| `expected_social_return` | Expected return or benefit measure |
| `financing_cost` | Cost of financing investment |
| `investment_affordability_gap` | Return minus financing cost |
