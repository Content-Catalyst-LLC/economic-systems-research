# Data Dictionary

| Variable | Description |
|---|---|
| `sector` | Emissions or transition sector |
| `output` | Economic activity or output proxy |
| `energy_intensity` | Energy use per unit output |
| `carbon_intensity` | Carbon emissions per unit energy |
| `emissions` | Output multiplied by energy intensity and carbon intensity |
| `old_emissions_intensity` | Baseline emissions intensity |
| `new_emissions_intensity` | Current or policy scenario emissions intensity |
| `years` | Transition period |
| `decarbonization_rate` | Annual change in emissions intensity |
| `carbon_price_strength` | Relative strength of carbon-pricing policy |
| `regulatory_strength` | Relative strength of standards, mandates, phaseouts, and performance rules |
| `public_investment` | Public investment in transition infrastructure and capacity |
| `industrial_policy` | Strategic clean-industry capability and procurement support |
| `social_protection` | Household, worker, and regional protection during transition |
| `transition_investment_score` | Composite investment-enabling score |
| `lock_in_score` | Composite carbon-lock-in score |
| `damage_risk_score` | Composite physical climate-damage risk |
| `just_transition_score` | Composite labor and regional fairness score |
| `global_equity_score` | Composite international burden-sharing and cooperation score |
| `implementation_credibility_score` | Composite policy durability and administrative capacity score |
