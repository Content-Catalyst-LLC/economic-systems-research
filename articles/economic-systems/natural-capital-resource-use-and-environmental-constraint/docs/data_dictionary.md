# Data Dictionary

| Variable | Description |
|---|---|
| `system` | Natural-capital or resource system |
| `natural_capital_t` | Current-period natural-capital stock |
| `regeneration` | Regeneration or replenishment flow |
| `degradation` | Depletion or degradation flow |
| `natural_capital_next` | Next-period natural-capital stock |
| `regeneration_gap` | Degradation minus regeneration |
| `resource_use` | Resource extraction or use flow |
| `regenerative_capacity` | Renewable-system capacity to regenerate |
| `resource_use_ratio` | Use divided by regenerative capacity |
| `emissions` | Pollution or waste flow |
| `absorptive_capacity` | Sink capacity to absorb waste safely |
| `waste_constraint_ratio` | Emissions divided by absorptive capacity |
| `material_intensity` | Material dependence of a sector or system |
| `energy_intensity` | Energy dependence of a sector or system |
| `import_dependence` | External dependence on imported resource flows |
| `extraction_dependency_score` | Composite material-energy-import vulnerability |
| `exposure` | Exposure to ecological burden |
| `income_buffer` | Ability to absorb ecological stress through income/assets |
| `public_infrastructure` | Public infrastructure protection and capacity |
| `adaptive_capacity` | Ability to adapt, recover, relocate, insure, or substitute |
| `justice_burden_score` | Composite ecological burden score |
| `resilience_score` | Composite ecological resilience score |
