# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Resilience, fragility, or shock scenario |
| `buffers` | Spare financial, institutional, or material capacity |
| `redundancy` | Backup capacity and alternative pathways |
| `coordination` | Institutional coordination capacity |
| `trust` | Social and institutional trust |
| `learning` | Ability to learn from disruption |
| `resilience_score` | Composite resilience score |
| `leverage` | Financial leverage or debt dependence |
| `concentration` | Supplier, sector, asset, or ownership concentration |
| `exposure` | Degree of exposure to shocks |
| `underinvestment` | Maintenance, public goods, or infrastructure underinvestment |
| `inequality` | Distributional vulnerability |
| `fragility_score` | Composite fragility score |
| `information` | Data, warning, and situational-awareness capacity |
| `fiscal_space` | Public fiscal capacity to respond |
| `skills` | Workforce skill and retraining capacity |
| `flexibility` | Ability to reallocate resources and adapt |
| `legitimacy` | Public legitimacy of response institutions |
| `adaptive_capacity_score` | Composite adaptive-capacity score |
| `shock_magnitude` | Severity of disturbance |
| `vulnerability` | System vulnerability to disturbance |
| `shock_impact` | Shock magnitude multiplied by vulnerability |
| `recovery_capacity` | Composite recovery capacity |
