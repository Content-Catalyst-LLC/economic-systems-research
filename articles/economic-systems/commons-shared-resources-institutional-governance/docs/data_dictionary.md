# Data Dictionary

| Variable | Description |
|---|---|
| `resource` | Shared resource type |
| `scenario` | Resource or governance scenario |
| `initial_stock` | Starting stock of the shared resource |
| `carrying_capacity` | Maximum stock level supported by the system |
| `regeneration_rate` | Natural regeneration parameter |
| `critical_stock_threshold` | Stock level below which breakdown or depletion risk rises |
| `user_group` | Group drawing from or governing the shared resource |
| `baseline_extraction` | Extraction under weak or normal governance |
| `governed_extraction` | Extraction under stronger governance or restraint |
| `access_rights_index` | Degree of recognized access rights |
| `compliance_tendency` | Baseline willingness or capacity to comply |
| `monitoring_capacity` | Capacity to observe use and conditions |
| `rule_clarity` | Clarity of governance rules |
| `legitimacy` | Perceived legitimacy of governance |
| `sanction_capacity` | Capacity to enforce consequences for rule breach |
| `capture_risk` | Risk that powerful actors capture governance |
| `local_knowledge_use` | Degree to which governance uses situated knowledge |
| `polycentric_coordination` | Degree of nested coordination across scales |
| `institutional_upkeep` | Investment in maintenance, monitoring, conflict resolution, and legitimacy |
| `institutional_erosion` | Erosion from conflict, neglect, capture, or rule breakdown |
| `stock` | Resource stock at a given time |
| `regeneration` | Resource regeneration at a given stock |
| `harvest` | Aggregate extraction by users |
| `compliance_score` | Composite compliance indicator |
| `justice_score` | Composite indicator balancing preservation, access, and fairness |
| `institutional_integrity` | Simulated institutional-capacity stock |
