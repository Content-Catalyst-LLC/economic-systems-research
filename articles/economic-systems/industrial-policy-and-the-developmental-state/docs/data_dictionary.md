# Data Dictionary

| Variable | Description |
|---|---|
| `sector` | Strategic sector or industrial activity |
| `sector_output` | Output generated by sector |
| `total_output` | Total output across economy or comparison group |
| `sector_labor` | Labor employed in sector |
| `sector_exports` | Exports from sector |
| `public_support` | Direct or indirect public support |
| `productivity_gain` | Productivity improvement associated with policy support |
| `employment_gain` | Employment increase associated with policy support |
| `local_supplier_share` | Share of inputs sourced from domestic or local suppliers |
| `emissions_reduction` | Emissions reduction associated with industrial strategy |
| `support_intensity` | Public support divided by sector output |
| `export_ratio` | Sector exports divided by sector output |
| `sector_productivity` | Sector output divided by labor |
| `performance_score` | Composite score linking public support to outcomes |
| `patient_credit_share` | Share of finance with long-horizon developmental terms |
| `speculative_credit_share` | Share of finance going to speculative or non-transformative uses |
| `infrastructure_reliability` | Infrastructure readiness score |
| `energy_reliability` | Energy-system reliability score |
| `skills_alignment` | Alignment between skills systems and industrial needs |
| `capture_risk_score` | Composite risk of rent-seeking and capture |
| `cluster_depth_score` | Composite regional cluster strength |
| `green_industrial_score` | Composite green industrial-policy performance |
