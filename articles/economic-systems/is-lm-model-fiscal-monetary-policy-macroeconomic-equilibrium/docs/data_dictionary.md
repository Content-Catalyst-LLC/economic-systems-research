# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario name |
| `alpha` | Autonomous demand parameter |
| `beta` | Interest sensitivity of investment / goods-market demand |
| `gamma` | Interest-rate response to output through money-market equilibrium |
| `fiscal_shift` | Exogenous shift in goods-market demand |
| `money_shift` | Monetary-policy shift affecting LM position |
| `equilibrium_output` | Solved IS–LM equilibrium output |
| `equilibrium_interest_rate` | Solved IS–LM equilibrium interest rate |
| `delta_output_from_baseline` | Output change relative to baseline |
| `delta_interest_from_baseline` | Interest-rate change relative to baseline |
| `fiscal_multiplier_model` | Change in output per unit fiscal shift under model parameters |
| `monetary_multiplier_model` | Change in output per unit money shift under model parameters |
| `crowding_out_indicator` | Interest-rate increase under fiscal expansion |
| `lm_slope_type` | Flat, baseline, or steep LM case |
