# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Mixed-economy or policy scenario |
| `market_allocation` | Share or intensity of market coordination |
| `public_planning` | Share or intensity of public planning |
| `public_provision` | Share or intensity of public service/provision |
| `regulation_rights` | Regulation and social-rights intensity |
| `public_ownership_share` | Public-sector output divided by total output |
| `state_capacity` | Administrative and strategic state capacity |
| `data_quality` | Quality of planning information and measurement |
| `institutional_reach` | Ability of institutions to implement policy across territory/sectors |
| `feedback_quality` | Learning, monitoring, correction, and local feedback quality |
| `democratic_accountability` | Public oversight, contestability, transparency, and participation |
| `planning_capacity_score` | Composite planning capacity score |
| `decommodification_score` | Composite public services, social insurance, and guaranteed access score |
| `sector_fit_market` | Suitability of market allocation in a sector |
| `sector_fit_planning` | Suitability of public planning/coordination in a sector |
| `utility_public_purpose_score` | Universal access, investment, affordability, and accountability score |
| `industrial_policy_score` | Strategic coordination, public investment, learning, discipline, and upgrading score |
| `crisis_coordination_score` | Public planning, infrastructure depth, fiscal capacity, and administrative speed score |
| `globalization_constraint_score` | External finance, import dependence, multinational power, and legal constraints score |
| `sustainable_transition_score` | Public investment, social protection, ecological targets, and legitimacy score |
