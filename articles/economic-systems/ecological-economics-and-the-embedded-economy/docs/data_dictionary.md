# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Ecological-economy scenario name |
| `energy_input` | Energy entering the economic system |
| `material_input` | Material entering the economic system |
| `throughput` | Energy input plus material input |
| `recovered_throughput` | Recovered, recycled, reused, or regenerated flow |
| `waste_residual` | Throughput minus recovered throughput |
| `economic_scale` | Stylized scale of economic activity |
| `ecological_capacity` | Stylized regenerative or absorptive capacity |
| `scale_ratio` | Economic scale divided by ecological capacity |
| `overshoot_risk` | Risk of exceeding ecological capacity |
| `domestic_extraction` | Domestic material extraction |
| `imports` | Imported material flows |
| `exports` | Exported material flows |
| `material_footprint` | Domestic extraction plus imports minus exports |
| `exposure` | Exposure to ecological hazard |
| `income` | Household, group, or territorial income proxy |
| `infrastructure` | Protective infrastructure and public-service capacity |
| `adaptive_capacity` | Ability to absorb, adapt, relocate, insure, or recover |
| `ecological_burden_score` | Composite ecological burden score |
| `care_capacity` | Capacity of care systems and social reproduction |
| `institutional_trust` | Trust and functionality of public institutions |
| `embeddedness_score` | Composite social and ecological embeddedness score |
| `resilience_score` | Composite diversity, redundancy, regeneration, governance, maintenance, and learning score |
