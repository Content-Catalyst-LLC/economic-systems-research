# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Externality or public-good scenario |
| `output` | Output level |
| `MPC` | Marginal private cost |
| `MEC` | Marginal external cost |
| `MSC` | Marginal social cost |
| `MPB` | Marginal private benefit |
| `MEB` | Marginal external benefit |
| `MSB` | Marginal social benefit |
| `private_output` | Output selected when private benefit equals private cost |
| `social_output` | Output selected when social benefit equals social cost |
| `overproduction_gap` | Private output minus social output for negative externalities |
| `underprovision_gap` | Social need minus private or voluntary provision |
| `contributor_group` | Household, firm, region, or institutional contributor group |
| `contribution` | Contribution to the public good |
| `benefit_share` | Share of public-good benefit received |
| `free_rider_flag` | Indicator for benefiting with little or no contribution |
| `tax_revenue` | Revenue used for collective provision |
| `borrowing_capacity` | Intertemporal financing capacity |
| `pooled_funds` | Other organized contribution or pooled institutional finance |
| `collective_provision_capacity` | Total capacity to fund shared provision |
| `burden_share` | Share of external costs borne by a group |
| `damage` | Accumulated damage indicator |
| `critical_threshold` | Threshold beyond which recovery becomes difficult or impossible |
| `threshold_crossed` | Indicator for crossing the threshold |
