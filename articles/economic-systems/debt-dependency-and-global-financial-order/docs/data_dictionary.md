# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `debt_stock` | Total debt stock |
| `output` | Total output or income |
| `debt_ratio` | Debt stock divided by output |
| `external_debt_service` | External debt service due |
| `export_earnings` | Export earnings available for FX obligations |
| `external_debt_service_ratio` | External debt service divided by export earnings |
| `foreign_currency_debt` | Debt denominated in foreign currency |
| `exchange_rate` | Domestic currency per foreign currency unit |
| `domestic_currency_fx_burden` | Foreign-currency debt times exchange rate |
| `effective_interest_rate` | Average interest rate on debt |
| `growth_rate` | Output growth rate |
| `primary_deficit` | Primary deficit as share or value |
| `reserves` | Foreign-exchange reserves |
| `short_term_external_obligations` | Short-term external obligations due |
| `reserve_adequacy` | Reserves divided by short-term external obligations |
| `import_dependence` | Dependence on imported essentials and capital goods |
| `capital_flow_volatility` | Volatility of capital flows |
| `external_finance_reliance` | Reliance on external finance |
| `policy_space` | Fiscal, monetary, or developmental policy-space score |
| `austerity_intensity` | Strength of fiscal compression |
| `public_capacity_erosion` | Decline in health, education, infrastructure, or administrative capacity |
| `restructuring_haircut` | Reduction in debt principal |
| `maturity_extension` | Additional maturity or grace period |
| `interest_relief` | Reduction in effective interest burden |
| `sustainable_finance_score` | Composite score for finance that builds future capacity |
