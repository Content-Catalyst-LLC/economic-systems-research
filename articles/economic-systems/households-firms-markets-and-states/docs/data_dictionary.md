# Data Dictionary

| Variable | Description |
|---|---|
| `actor` | Institutional actor: household, firm, market, or state |
| `core_role` | Main role in the economic system |
| `main_inputs` | Resources or capacities the actor depends on |
| `main_outputs` | Goods, services, income, rules, coordination, or stability the actor contributes |
| `risk_absorption_role` | Type of risk the actor absorbs or shifts |
| `household_wages` | Household labor income |
| `household_transfers` | Public or institutional transfers to households |
| `asset_income` | Income from financial or real assets |
| `taxes_paid` | Household tax payments |
| `debt_service` | Household debt-service burden |
| `household_consumption` | Household consumption expenditure |
| `household_saving` | Household income remaining after consumption, taxes, and debt service |
| `firm_revenue` | Firm revenue from sales |
| `labor_cost` | Firm labor-cost payments |
| `capital_cost` | Financing or capital-service cost |
| `input_cost` | Material and intermediate input cost |
| `firm_profit` | Revenue less labor, capital, and input costs |
| `public_spending` | State spending on goods, services, infrastructure, or public provision |
| `public_transfers` | Transfers from state to households or institutions |
| `public_debt_interest` | Interest paid on public debt |
| `tax_revenue` | State tax revenue |
| `public_borrowing` | Public borrowing required under the scenario |
| `effective_access` | Access score based on income, institutional provision, need, and prices |
| `reproduction_score` | Composite score for household, productive, and institutional reproduction |
