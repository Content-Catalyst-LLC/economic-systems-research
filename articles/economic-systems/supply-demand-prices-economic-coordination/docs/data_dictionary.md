# Data Dictionary

| Variable | Description |
|---|---|
| `market` | Market or sector name |
| `scenario` | Scenario name |
| `a` | Demand intercept |
| `b` | Demand slope; demand responsiveness to price |
| `c` | Supply intercept |
| `d` | Supply slope; supply responsiveness to price |
| `external_cost` | Marginal external cost not reflected in private price |
| `markup_rate` | Stylized markup parameter for market-power scenarios |
| `equilibrium_price` | Market-clearing price |
| `equilibrium_quantity` | Market-clearing quantity |
| `price_change_from_baseline` | Change in price relative to baseline |
| `quantity_change_from_baseline` | Change in quantity relative to baseline |
| `demand_elasticity` | Point elasticity of demand at equilibrium |
| `supply_elasticity` | Point elasticity of supply at equilibrium |
| `marginal_social_cost_price` | Private price plus external cost |
| `social_cost_gap` | Difference between social-cost price and private market price |
| `need_index` | Underlying need measure |
| `income_command` | Purchasing power measure |
| `credit_access` | Access to borrowing or liquidity |
| `institutional_access` | Access through public provision, entitlement, or infrastructure |
| `effective_demand` | Need filtered through income, credit, prices, and access |
| `unmet_need_gap` | Need minus effective demand/access |
