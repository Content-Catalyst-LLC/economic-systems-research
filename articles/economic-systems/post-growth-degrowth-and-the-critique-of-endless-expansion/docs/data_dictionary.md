# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Post-growth or degrowth scenario |
| `employment_dependency` | Degree to which employment stability depends on growth |
| `debt_service_dependency` | Degree to which debt service depends on future growth |
| `fiscal_dependency` | Degree to which public budgets depend on growth |
| `asset_dependency` | Degree to which asset valuations depend on expansion |
| `growth_dependence_score` | Composite growth dependence indicator |
| `population` | Population index |
| `affluence` | Output/consumption per person index |
| `intensity` | Material or energy intensity |
| `throughput_index` | Population times affluence times intensity |
| `health` | Health index |
| `time_balance` | Time balance index |
| `security` | Economic and social security index |
| `equality` | Equality/distribution index |
| `public_goods` | Public goods and services index |
| `ecological_quality` | Ecological quality index |
| `wellbeing_score` | Composite wellbeing score |
| `needs_met` | Share of core needs met |
| `throughput_required` | Throughput required to meet needs |
| `sufficiency_score` | Needs met divided by throughput required |
| `degrowth_transition_score` | Composite transition credibility score |
