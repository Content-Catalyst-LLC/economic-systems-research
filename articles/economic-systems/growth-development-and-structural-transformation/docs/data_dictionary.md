# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `period` | Time period |
| `output` | Aggregate output |
| `labor` | Labor input |
| `growth_rate` | Period-to-period output growth |
| `labor_productivity` | Output divided by labor |
| `sector` | Economic sector |
| `sector_output` | Output by sector |
| `sector_labor` | Labor by sector |
| `output_share` | Sector output divided by total output |
| `labor_share` | Sector labor divided by total labor |
| `sector_productivity` | Sector output divided by sector labor |
| `income_index` | Normalized income indicator |
| `health_index` | Normalized health indicator |
| `education_index` | Normalized education indicator |
| `infrastructure_index` | Normalized infrastructure indicator |
| `security_index` | Normalized security indicator |
| `capability_index` | Multidimensional development index |
| `urbanization_rate` | Share of population in urban areas |
| `infrastructure_depth` | Composite infrastructure quality indicator |
| `spatial_inclusion` | Territorial inclusion score |
| `export_concentration` | Concentration of export basket |
| `technology_depth` | Domestic technological or industrial capability score |
| `gini_index` | Inequality measure |
| `informal_labor_share` | Share of labor in informal work |
| `energy_use` | Energy use indicator |
| `emissions` | Emissions indicator |
| `energy_intensity` | Energy use divided by output |
| `external_debt_ratio` | External debt relative to output or exports |
| `fx_debt_share` | Foreign-currency debt share |
| `middle_income_fragility_score` | Composite fragility score |
