# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Fiscal scenario identifier |
| `tax_revenue` | Public tax revenue |
| `public_spending` | Total public spending |
| `primary_spending` | Public spending excluding interest |
| `interest_rate` | Effective interest rate on public debt |
| `debt_stock` | Initial public debt stock |
| `output` | Output or income base |
| `budget_balance` | Tax revenue minus public spending |
| `primary_balance` | Tax revenue minus primary spending |
| `tax_ratio` | Tax revenue divided by output |
| `debt_to_output` | Public debt divided by output |
| `next_period_debt` | Debt after deficit and interest costs |
| `income_group` | Household or tax-unit income group |
| `income` | Income before tax |
| `tax_paid` | Taxes paid |
| `effective_tax_rate` | Tax paid divided by income |
| `transfer_received` | Public transfers received |
| `net_fiscal_position` | Transfers minus taxes or taxes minus transfers depending on table |
| `spending_category` | Type of public spending |
| `spending_amount` | Spending level by category |
| `public_investment_share` | Public investment divided by total spending |
| `delta_g` | Change in spending |
| `fiscal_multiplier` | Output response parameter |
| `delta_y` | Estimated output effect |
| `maintenance_needed` | Required maintenance spending |
| `maintenance_actual` | Actual maintenance spending |
| `maintenance_gap` | Needed minus actual maintenance |
| `tax_base_per_capita` | Local taxable base per resident |
| `service_need_index` | Relative index of local public service need |
| `fiscal_capacity_score` | Composite local capacity score |
| `public_investment` | Public investment amount |
| `avoided_future_losses` | Estimated avoided future loss |
| `resilience_score` | Composite resilience score |
| `avoided_loss_return` | Avoided losses divided by investment |
