# Data Dictionary

| Variable | Description |
|---|---|
| `regime` | Capitalist regime or scenario name |
| `revenue` | Firm or sector revenue |
| `cost` | Firm or sector cost |
| `profit` | Revenue minus cost |
| `wages` | Total wage income |
| `output` | Total output or income |
| `wage_share` | Wages divided by output |
| `profit_share` | Profit divided by output |
| `finance_type` | Degree of patient versus arm's-length finance |
| `labor_regime` | Labor coordination, bargaining, or protection score |
| `training_system` | Skill-formation and vocational capacity score |
| `firm_coordination` | Coordination among firms, suppliers, banks, workers, and public authority |
| `financialization_score` | Composite asset-price, leverage, shareholder-claim, and debt intensity |
| `welfare_buffer_score` | Composite transfers, services, coverage, and replacement-rate strength |
| `housing_vulnerability` | Housing-cost, debt, rent burden, and asset-price pressure |
| `corporate_governance_score` | Governance time horizon, stakeholder coordination, and investment orientation |
| `innovation_score` | Regime-specific innovation capacity |
| `crisis_response_score` | Automatic stabilizers, public capacity, restructuring, and household buffering |
| `institutional_advantage_score` | Comparative institutional advantage score |
| `sustainable_capitalism_score` | Productive investment, labor security, welfare buffering, ecological constraint, and legitimacy |
