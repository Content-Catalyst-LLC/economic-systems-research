# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Beyond-GDP measurement scenario |
| `consumption` | Household/private consumption |
| `investment` | Investment |
| `government` | Government spending |
| `net_exports` | Exports minus imports |
| `gdp` | Consumption plus investment plus government spending plus net exports |
| `health` | Health outcome index |
| `education` | Education/capability index |
| `income_security` | Income security index |
| `housing` | Housing security/affordability index |
| `safety` | Safety and security index |
| `social_connection` | Social connection/trust index |
| `environment` | Environmental quality index |
| `time_balance` | Time availability and work-life balance index |
| `wellbeing_score` | Composite well-being score |
| `distribution` | Distributional fairness index |
| `mobility` | Social mobility index |
| `access` | Access to services and opportunity |
| `voice` | Political/social voice index |
| `inclusion_score` | Composite inclusion score |
| `natural_capital` | Natural-capital stock/quality index |
| `human_capital` | Human capability and skill index |
| `institutional_trust` | Trust and institutional quality index |
| `produced_capital` | Built/infrastructure capital index |
| `sustainability_score` | Composite sustainability score |
| `social_costs` | Social costs or exclusion costs |
| `ecological_costs` | Ecological depletion/damage costs |
| `adjusted_progress` | Current benefits minus social and ecological costs |
