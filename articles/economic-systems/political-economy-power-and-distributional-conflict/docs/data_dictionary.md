# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `wages` | Wage income |
| `profits` | Profit income |
| `rents` | Rent income |
| `total_income` | Sum of wages, profits, and rents |
| `wage_share` | Wages divided by total income |
| `profit_share` | Profits divided by total income |
| `rent_share` | Rents divided by total income |
| `taxes_paid` | Taxes paid by group |
| `benefits_received` | Benefits and public transfers received by group |
| `net_fiscal_position` | Benefits received minus taxes paid |
| `ownership_power` | Economic power derived from asset ownership |
| `organization_power` | Power derived from collective organization |
| `access_power` | Access to policymaking, administration, media, or legal systems |
| `mobility_power` | Exit options and ability to relocate capital, labor, or claims |
| `voice_power` | Ability to influence decisions through representation and contestation |
| `power_asymmetry_score` | Composite power score |
| `inequality_pressure` | Material inequality pressure |
| `inflation_pressure` | Price stress and real-income conflict |
| `unemployment_pressure` | Labor-market stress |
| `representation_gap` | Gap between affected groups and institutional voice |
| `shock_exposure` | Exposure to crisis, inflation, debt, climate, or labor-market shocks |
| `conflict_intensity_score` | Composite distributional-conflict score |
| `legitimacy_score` | Fairness, security, voice, institutional trust, and public credibility |
