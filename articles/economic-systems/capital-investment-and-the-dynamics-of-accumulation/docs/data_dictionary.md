# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `initial_capital` | Starting capital stock |
| `investment` | Gross investment during period |
| `depreciation_rate` | Share of capital stock depreciated per period |
| `labor` | Labor input or employment level |
| `output` | Output or value added |
| `capital_stock` | Modeled capital stock at a given period |
| `capital_intensity` | Capital stock divided by labor |
| `investment_rate` | Investment divided by output |
| `project` | Candidate investment project |
| `cashflow_year_0` ... `cashflow_year_5` | Project cash-flow sequence |
| `private_discount_rate` | Discount rate used by private investors |
| `public_discount_rate` | Discount rate used for public or social appraisal |
| `npv_private` | Net present value under private discounting |
| `npv_public` | Net present value under public discounting |
| `new_maintenance_need` | Newly arising maintenance need |
| `actual_maintenance` | Maintenance actually performed |
| `maintenance_backlog` | Deferred maintenance stock |
| `claimant` | Wages, profits, taxes, rents, or public reinvestment |
| `claim_share` | Share of output or surplus captured by claimant |
| `finance_channel` | Productive investment, speculation, extraction, public infrastructure, resilience, or maintenance |
| `flow_share` | Share of finance directed to a channel |
| `productive_capacity_score` | Score for future productive capacity |
| `resilience_score` | Score for resilience and robustness |
| `ecological_alignment_score` | Score for ecological sustainability |
| `public_value_score` | Score for public or social value |
| `sustainable_investment_score` | Composite sustainable-investment score |
