# Data Dictionary

| Variable | Description |
|---|---|
| `date` | Observation date |
| `recession_indicator` | NBER-based recession indicator from FRED |
| `constraint_phase` | Expansion or recession phase label |
| `unemployment_rate` | Civilian unemployment rate |
| `federal_funds_rate` | Effective federal funds rate |
| `treasury_bill_3m` | 3-month Treasury bill rate |
| `treasury_10y` | 10-year Treasury constant maturity rate |
| `cpi` | Consumer price index |
| `real_gdp` | Real gross domestic product |
| `potential_gdp` | Real potential GDP |
| `debt_to_gdp` | Federal debt as percent of GDP |
| `real_gdp_growth_annualized` | Annualized quarterly real GDP growth |
| `output_gap_pct` | Percent gap between real GDP and potential GDP |
| `cpi_inflation_yoy_pct` | Year-over-year CPI inflation |
| `delta_unemployment_rate` | Quarterly change in unemployment rate |
| `real_interest_rate_proxy` | 10-year Treasury rate minus year-over-year inflation |
| `interest_growth_gap` | Nominal long-term interest-rate proxy minus nominal GDP growth proxy |
| `debt_stabilizing_primary_balance_pct_gdp` | Simplified debt-stabilizing balance proxy |
| `lower_bound_constraint_flag` | Indicator for low federal funds rate periods |
| `inflation_constraint_flag` | Indicator for elevated inflation and non-negative output gap |
| `fiscal_space_constraint_flag` | Indicator for elevated debt and positive interest-growth gap |
| `crowding_out_proxy` | Proxy combining debt ratio, long-term rates, and output-gap conditions |
