# Data Dictionary

| Variable | Description |
|---|---|
| `institution` | Financial institution, fund, household segment, or system actor |
| `assets` | Total assets |
| `debt` | Total debt or fixed liabilities |
| `equity` | Assets minus debt |
| `leverage` | Assets divided by equity |
| `debt_to_equity` | Debt divided by equity |
| `asset_shock` | Percentage decline in asset value |
| `equity_after_shock` | Remaining equity after asset shock |
| `equity_loss_pct` | Percent change in equity after shock |
| `collateral_value` | Market value of pledged collateral |
| `haircut_before` | Initial collateral haircut |
| `haircut_after` | Stressed collateral haircut |
| `borrowing_capacity_before` | Borrowing capacity before haircut shock |
| `borrowing_capacity_after` | Borrowing capacity after haircut shock |
| `funding_gap` | Shortfall created by reduced borrowing capacity or maturing liabilities |
| `short_term_liabilities` | Near-term liabilities coming due |
| `liquid_assets` | Immediately usable liquid assets |
| `fire_sale_sales` | Forced asset sales in period |
| `price_index` | Asset price index during fire-sale loop |
| `network_exposure` | Exposure from one institution to another |
| `capital_buffer` | Loss-absorbing capital buffer |
| `mortgage_debt` | Household mortgage debt |
| `housing_value` | Housing asset value |
| `loan_to_value` | Mortgage debt divided by housing value |
| `macroprudential_buffer` | Countercyclical or systemic-risk buffer |
| `sustainability_shock` | Climate, insurance, infrastructure, or stranded-asset stress |
