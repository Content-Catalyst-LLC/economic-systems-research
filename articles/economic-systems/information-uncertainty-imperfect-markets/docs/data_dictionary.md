# Data Dictionary

| Variable | Description |
|---|---|
| `type` | High-quality, low-quality, safe, risky, or other hidden type |
| `share` | Share of market participants or products with a given type |
| `value` | Buyer, lender, or social value associated with type |
| `reservation_value` | Minimum value required for a type to remain in the market |
| `risk_probability` | Probability of default, loss, or failure |
| `loss_given_event` | Loss conditional on adverse event |
| `effort` | Hidden effort level chosen after agreement |
| `output` | Output generated by effort |
| `effort_cost` | Cost of effort to agent |
| `insured_share` | Share of downside risk shifted away from agent |
| `signal` | Warranty, credential, audit, certification, collateral, or other signal |
| `signal_cost_high_type` | Cost of signal for high-quality type |
| `signal_cost_low_type` | Cost of signal for low-quality type |
| `credibility_gap` | Cost gap that makes imitation difficult |
| `information_level` | Amount of search, audit, inspection, disclosure, or verification |
| `benefit_information` | Benefit from improved information |
| `cost_information` | Acquisition or processing cost of information |
| `net_information_value` | Benefit less cost of information |
| `action` | Candidate decision under uncertainty |
| `good_case` | Payoff under favorable scenario |
| `moderate_case` | Payoff under moderate scenario |
| `bad_case` | Payoff under adverse scenario |
| `worst_case` | Minimum payoff across scenarios |
| `complexity_index` | Contract/product/system complexity |
| `hidden_fee_index` | Opaque fee or penalty exposure |
| `trust_index` | Trust or credibility level |
| `effective_intelligibility` | Modeled ease of meaningful understanding |
