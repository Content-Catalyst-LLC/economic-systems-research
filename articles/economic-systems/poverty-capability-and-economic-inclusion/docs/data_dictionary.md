# Data Dictionary

| Variable | Description |
|---|---|
| `household` | Household or scenario identifier |
| `income` | Household income or consumption |
| `poverty_line` | Poverty threshold |
| `poverty_status` | Whether household lies below the threshold |
| `poverty_gap` | Shortfall from poverty line, divided by population or threshold depending on context |
| `health_deprivation` | Health deprivation indicator |
| `education_deprivation` | Education deprivation indicator |
| `housing_deprivation` | Housing deprivation indicator |
| `sanitation_deprivation` | Sanitation deprivation indicator |
| `food_deprivation` | Nutrition or food security deprivation |
| `transport_deprivation` | Mobility or transport deprivation |
| `digital_deprivation` | Broadband, device, or digital-literacy deprivation |
| `institutional_access` | Ability to access services, identification, benefits, and legal systems |
| `capability_score` | Composite capability score |
| `conversion_condition_score` | Score showing how effectively resources become practical freedom |
| `inclusion_score` | Composite economic inclusion score |
| `vulnerability_score` | Composite vulnerability score |
| `work_security` | Stability and protection of work |
| `informality_risk` | Degree of informal or unprotected employment |
| `housing_security` | Housing stability and adequacy |
| `financial_inclusion_quality` | Whether finance supports capability rather than extraction |
| `care_burden` | Time and labor burden of unpaid care |
| `public_service_score` | Public services and social-protection strength |
| `shock_exposure` | Exposure to illness, job loss, climate, rent, food, or debt shocks |
