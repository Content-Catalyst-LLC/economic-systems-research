# Economic Resilience: Why Recessions Occur and How Economies Recover

This article companion folder supports the Economic Systems article:

**Economic Resilience: Why Recessions Occur and How Economies Recover**

The standardized Economic Systems research stack is:

- Python
- R
- Stata
- SQL
- Julia

## Purpose

This article explains why recessions occur, how demand collapse can create involuntary unemployment, how economies recover, and why economic resilience depends on institutions, public capacity, financial stability, household protection, and recovery quality.

## Workflow

1. Use Python to download and prepare macroeconomic indicators.
2. Use R to summarize recession and recovery behavior.
3. Use Stata for applied econometric workflows.
4. Use SQL to define transparent indicator tables and queries.
5. Use Julia for future dynamic recession and recovery simulations.
