# Data Dictionary

| Variable | Description |
|---|---|
| `boundary` | Earth-system process or boundary domain |
| `economic_pressure` | Stylized human/economic pressure on the boundary |
| `earth_system_capacity` | Stylized safe operating capacity |
| `boundary_pressure_ratio` | Economic pressure divided by Earth-system capacity |
| `overshoot_gap` | Boundary pressure ratio minus one |
| `irreversibility` | Risk of difficult-to-reverse damage |
| `system_connectivity` | Degree of linkage to other Earth-system processes |
| `population` | Population index |
| `affluence` | Output/consumption per person index |
| `resource_intensity` | Resources used per unit affluence |
| `resource_use` | Population times affluence times resource intensity |
| `sector_pressure_score` | Composite sector pressure across boundary domains |
| `justice_adjusted_sustainability` | Composite ecological stability, inclusion, and development capability score |
| `transition_capacity_score` | Composite capacity to implement boundary-aware transition |
| `finance_direction_score` | Composite score for finance aligned with safe operating space |
| `boundary_aware_progress` | Wellbeing minus overshoot and exclusion penalties |
