# Data Dictionary

| Variable | Description |
|---|---|
| `option_id` | Choice option identifier |
| `option_label` | Human-readable option name |
| `utility` | Baseline utility value |
| `complexity` | Cognitive or administrative complexity burden |
| `salience` | Visibility or attention value |
| `search_order` | Order in which options are encountered |
| `satisficing_threshold` | Utility threshold for accepting an option |
| `beta` | Present-bias parameter |
| `delta` | Standard discount factor |
| `period` | Time period |
| `future_value` | Future payoff |
| `outcome` | Gain or loss relative to reference point |
| `lambda_loss` | Loss-aversion coefficient |
| `alpha_gain` | Curvature for gains |
| `beta_loss` | Curvature for losses |
| `probability` | Objective probability |
| `weighted_probability` | Psychologically transformed probability |
| `frame` | Gain, loss, neutral, or social-comparison frame |
| `default_status` | Whether a beneficial option is the default |
| `benefit_value` | Program or option benefit |
| `admin_burden` | Administrative, cognitive, time, or documentation burden |
| `trust_index` | Trust in the institution or offer |
| `social_norm_signal` | Signal about peer behavior |
| `take_up_probability` | Modeled probability of uptake |
| `compliance_probability` | Modeled probability of compliance or cooperation |
