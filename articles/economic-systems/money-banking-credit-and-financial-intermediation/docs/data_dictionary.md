# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `currency` | Currency in circulation |
| `deposits` | Deposit money |
| `money_supply` | Currency plus deposits |
| `assets` | Bank assets |
| `equity` | Bank equity |
| `loans` | Bank loans |
| `bank_deposits` | Bank deposit liabilities |
| `liquid_assets` | High-quality liquid assets or liquid asset buffer |
| `leverage` | Assets divided by equity |
| `capital_ratio` | Equity divided by assets |
| `loan_to_deposit_ratio` | Loans divided by deposits |
| `new_loan` | New lending amount |
| `assets_after` | Assets after lending expansion |
| `deposits_after` | Deposits after lending expansion |
| `borrower_group` | Household, firm, government, or other borrower type |
| `income_or_ebit` | Income or earnings before interest and taxes |
| `interest_payment` | Interest payment burden |
| `principal_payment` | Principal payment burden |
| `interest_coverage_ratio` | Income or EBIT divided by interest |
| `debt_service_ratio` | Total debt service divided by income |
| `hqla` | High-quality liquid assets |
| `net_cash_outflows` | Net cash outflows over stress horizon |
| `liquidity_coverage_ratio` | HQLA divided by net cash outflows |
| `credit_channel` | Destination of credit or financial flows |
| `flow_share` | Share of credit allocation |
| `productive_score` | Productive-capacity contribution score |
| `speculation_score` | Asset speculation score |
| `resilience_score` | Financial or systems resilience score |
| `inclusion_score` | Inclusion and access score |
| `sustainable_finance_score` | Composite finance direction score |
