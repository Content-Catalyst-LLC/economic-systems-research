# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Future economic-system scenario |
| `ecological_pressure` | Ecological pressure index; lower is better |
| `institutional_capacity` | Ability of public and social institutions to coordinate transition |
| `social_inclusion` | Inclusion and distributional legitimacy index |
| `resilience` | Ability to absorb, recover, and adapt under stress |
| `future_viability_score` | Composite future viability indicator |
| `population` | Population index |
| `affluence` | Output or consumption per person index |
| `material_intensity` | Material or energy intensity of activity |
| `throughput_pressure` | Population times affluence times material intensity |
| `public_investment` | Public investment capacity |
| `policy_credibility` | Credibility and durability of transition policy |
| `technology` | Technological capability |
| `social_trust` | Social and institutional trust |
| `implementation_capacity` | Administrative ability to deliver transition |
| `transition_capacity_score` | Composite transition capacity score |
| `wellbeing_score` | Composite well-being beyond output score |
| `finance_direction_score` | Composite capital-allocation alignment score |
| `democratic_legitimacy_score` | Composite transition legitimacy score |
