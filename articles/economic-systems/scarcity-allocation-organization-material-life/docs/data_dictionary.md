# Data Dictionary

| Variable | Description |
|---|---|
| `priority` | Allocation category, such as housing, health, education, infrastructure, energy, care, or ecological restoration |
| `baseline_share` | Baseline share of total available resources |
| `alternative_share` | Alternative allocation share |
| `resilience_share` | Allocation scenario emphasizing resilience, maintenance, and restoration |
| `allocation_units` | Resource units allocated to a category |
| `change_from_baseline` | Change in allocation relative to baseline |
| `need_index` | Stylized index of need |
| `income_command` | Household purchasing-power measure |
| `price_index` | Cost or price barrier measure |
| `institutional_access` | Public provision, legal entitlement, or institutional access measure |
| `effective_access` | Combined access score based on need, income, price, and institutions |
| `deprivation_flag` | Indicator for low effective access relative to need |
| `consumption` | Present consumption allocation |
| `investment` | Future productive-capacity allocation |
| `maintenance` | Maintenance and repair allocation |
| `ecological_restoration` | Ecological restoration allocation |
| `produced_capacity` | Simulated produced capacity over time |
| `ecological_capacity` | Simulated ecological capacity over time |
| `resource_use` | Resource use or ecological burden |
| `regenerative_capacity` | Regeneration or absorptive capacity |
