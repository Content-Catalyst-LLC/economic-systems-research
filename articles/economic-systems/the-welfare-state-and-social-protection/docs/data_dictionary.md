# Data Dictionary

| Variable | Description |
|---|---|
| `scenario` | Scenario identifier |
| `market_income` | Income before taxes and transfers |
| `taxes` | Taxes paid |
| `transfers` | Cash or near-cash transfers |
| `disposable_income` | Market income minus taxes plus transfers |
| `social_spending` | Total social spending |
| `output` | Total output or income |
| `social_spending_ratio` | Social spending divided by output |
| `covered_population` | Population receiving or eligible for protection |
| `target_population` | Population meant to be protected |
| `coverage_rate` | Covered population divided by target population |
| `benefit` | Benefit amount |
| `previous_earnings` | Prior earnings before shock or retirement |
| `replacement_rate` | Benefit divided by previous earnings |
| `take_up_rate` | Share of eligible people actually receiving benefits |
| `administrative_burden` | Application, compliance, documentation, and appeal burden |
| `stigma_cost` | Social or psychological cost of claiming support |
| `vulnerability_score` | Composite vulnerability score |
| `protection_strength` | Composite strength of social protection |
| `care_burden` | Time and economic burden of unpaid care |
| `service_access` | Access to essential public services |
| `digital_access` | Ability to use digital administrative systems |
| `automation_error_risk` | Risk of incorrect denial, delay, or automated exclusion |
| `shock_exposure` | Exposure to unemployment, health, climate, inflation, or displacement shocks |
| `adaptive_scale_up_capacity` | Ability to expand protection during crises |
