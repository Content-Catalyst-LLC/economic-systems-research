# Data Dictionary

| Variable | Description |
|---|---|
| `group` | Income group, wealth group, household type, region, or scenario |
| `market_income` | Income before taxes and transfers |
| `taxes` | Tax payments |
| `transfers` | Cash or near-cash transfers |
| `public_service_value` | Estimated value of public services |
| `disposable_income` | Market income minus taxes plus transfers and service value |
| `wealth` | Net asset holdings |
| `debt` | Household liabilities |
| `wealth_ratio` | Wealth comparison between top and bottom groups |
| `income_share` | Group income divided by total income |
| `tax_rate` | Taxes divided by market income |
| `transfer_rate` | Transfers divided by market income |
| `redistributive_gain` | Disposable income minus market income |
| `housing_cost` | Housing cost or rent burden |
| `housing_cost_burden` | Housing cost divided by income |
| `education_access` | Education opportunity score |
| `health_access` | Health opportunity score |
| `place_advantage` | Neighborhood or regional opportunity score |
| `network_access` | Access to social and professional networks |
| `parent_outcome` | Parent status, income, education, or class indicator |
| `child_outcome` | Child status, income, education, or class indicator |
| `mobility_persistence` | Intergenerational persistence parameter |
| `bargaining_power` | Labor bargaining power score |
| `employment_security` | Employment stability score |
| `insurance_coverage` | Social-insurance protection score |
| `security_score` | Composite economic security score |
