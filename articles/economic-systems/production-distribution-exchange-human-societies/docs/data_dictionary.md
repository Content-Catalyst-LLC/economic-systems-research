# Data Dictionary

| Variable | Description |
|---|---|
| `sector_code` | Short sector identifier |
| `sector` | Economic sector name |
| `institutional_role` | Role of the sector in the wider economic system |
| `labor_share` | Share of sector output paid as labor income |
| `non_labor_share` | Share of sector output accruing as profit, rent, interest, or mixed income |
| `public_goods_dependence` | Stylized dependence on public goods, infrastructure, law, and state capacity |
| `ecological_intensity` | Stylized ecological throughput or pressure per unit of output |
| `employment_intensity` | Stylized employment requirement per unit of output |
| `trade_exposure` | Stylized exposure to exchange through trade, imports, exports, or external supply chains |
| `final_demand` | Final demand in each scenario |
| `total_output` | Total output required after input-output linkages |
| `indirect_output_requirement` | Total output minus final demand |
| `labor_income` | Total output multiplied by labor share |
| `non_labor_income` | Total output multiplied by non-labor share |
| `ecological_throughput` | Total output multiplied by ecological intensity |
| `employment_requirement` | Total output multiplied by employment intensity |
| `exchange_dependency` | Total output multiplied by trade exposure |
